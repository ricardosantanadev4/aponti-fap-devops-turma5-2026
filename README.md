# Turma DevOps

Repositório colaborativo da turma para prática de Git, GitHub, Branches, Commits, Pull Requests e GitHub Actions.

## Como participar

1. Faça um Fork do projeto
2. Crie uma branch
3. Adicione seu arquivo JSON na pasta `alunos`
4. Faça commit e push
5. Abra um Pull Request

Após o merge, seu nome será adicionado automaticamente à lista de integrantes.
